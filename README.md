## 📌 Project Overview

This project is a Flask-based Authentication System that allows users to register, log in, and securely access protected routes.

It demonstrates backend development concepts including user authentication, session management, and secure password handling.

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML
- CSS
- Jinja2 Templates
- SQLite / Database
- Render (Deployment)

---

## 🎯 Features

- User Registration
- User Login
- Secure Password Hashing
- Session Management
- Protected Dashboard Route
- Logout Functionality
- Deployed Online
